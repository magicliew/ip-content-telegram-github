## 起号诊断

### 为什么账号现在可能还没起号
1. **IP 记忆点需要更清楚：** 孙女 不能只是一个身份，要变成观众记得住的内容立场。
2. **受众要更精准：** 内容要明确讲给「家里有阿公阿嬷的年轻人」，不要让算法和观众都觉得太散。
3. **Hook 要更像心里话：** 开头不能只是记录生活，要像观众刷到时心里突然被讲中的一句话。
4. **留言点要更具体：** 不要只问『你觉得呢？』，要问观众很容易回答的真实生活细节。
5. **目前关键问题：** 很多账号不起号，不一定是内容不好，而是观众还没有一秒看懂：你是谁、你代表什么、为什么我要持续看你。

### 本周起号突破口
Week 先测试『把目标受众心里的那句话讲出来』，让 Hook 更像内心独白，而不是普通分享开场。


## 本周起号方向

Week 1 先用「孙女对阿公阿嬷」里的真实生活细节来建立 IP 记忆点。重点不是硬讲道理，而是把目标受众熟悉、但平时不会特别说出口的情绪讲出来。

这套内容适合测试共鸣型起号，因为它容易让观众留言、分享给同类受众，或 tag 身边的人。

### 标题 / Hook

原因：有具体关系冲突，适合短视频开头使用。

### 口播文案
嘴上说不用的人，其实最容易偷偷开心。

我以前一直觉得，孙女对阿公阿嬷之间，很多事情只要讲清楚就好了。可是后来才发现，不是每个人都会很直接地表达自己，尤其是我们家里那些比较传统、比较习惯忍一忍的人。

你买一包他平时舍不得买的东西，他第一句可能不是谢谢，而是：又乱花钱。可是隔天他可能会把东西收得好好的，甚至跟邻居讲：这是我孙女买的。

如果你家里也有这样的阿公阿嬷、爸爸妈妈，或者你自己就是很少把爱说出口的长辈，你可能也懂这种时刻：明明只是一件很小的事，嘴上没有夸，甚至还要念几句，可是心里其实是开心的。

这就是孙女这个角色最适合被记住的地方。不是每天讲大道理，也不是故意煽情，而是把这些很多家庭都有、但平时没说出口的细节讲出来。

嘴硬不是冷淡，很多时候是他们那一代人表达爱的方式比较含蓄。 所以这一篇想让观众看到：原来我家里那个人也是这样，原来不是只有我经历过这种感觉。

如果你家里也有一个嘴上不说、但行动很诚实的人，留言讲一个他的生活小习惯。
